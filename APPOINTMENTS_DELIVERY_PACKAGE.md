## 💡 USAGE GUIDE

### For Administrators

**Step 1: Navigate**
- Open admin dashboard
- Click "Appointments Management"

**Step 2: Filter**
- Click tabs for quick filters
- Use date range for specific dates
- Select priority or counselor
- Combine filters as needed

**Step 3: View**
- See all appointments in table
- Click View button for details
- Use Reset to start over

### For Developers

**Access Functions:**
```javascript
setupAppointmentFilters()           // Initialize
filterAndDisplayAppointments()      // Filter by type
applyAllFilters()                   // Combine filters
displayFilteredAppointments()       // Render
updateAppointmentsTabs()            // Update counts
resetAppointmentFilters()           // Reset
viewAppointmentDetails()            // Show details
```

**Extend Features:**
- Modify filter logic in `applyAllFilters()`
- Customize display in `displayFilteredAppointments()`
- Add new filter types in `filterAndDisplayAppointments()`
- Enhance calendar in `renderCalendarView()`
