## 🚀 How to Use

### For End Users (Administrators)

**Viewing All Appointments**
1. Click "Appointments Management" in the sidebar
2. See all appointments in list view

**Filtering by Tab**
1. Click any tab (Upcoming, Pending, Past, Missed)
2. Table updates instantly to show filtered appointments

**Filtering by Date Range**
1. Select start date in first date field
2. Select end date in second date field
3. Table filters to show appointments in that range

**Filtering by Priority**
1. Click priority dropdown
2. Select "High", "Medium", or "Low"
3. Table shows only that priority level

**Filtering by Counselor**
1. Click counselor dropdown
2. Select a counselor
3. Table shows only their appointments

**Combining Filters**
1. Select tab, date range, priority, and/or counselor
2. All filters work together
3. Table shows results matching all criteria

**Viewing Details**
1. Click the "View" button for any appointment
2. See full appointment details

**Resetting Filters**
1. Click "Reset" button
2. All filters clear
3. Table shows all appointments again

### For Developers

**Access filtered appointments:**
```javascript
filterAndDisplayAppointments('upcoming');  // Show upcoming
filterAndDisplayAppointments('pending');   // Show pending
filterAndDisplayAppointments('past');      // Show past
filterAndDisplayAppointments('missed');    // Show missed
```

**Apply custom filters:**
```javascript
applyAllFilters();  // Combines all active filters
```

**Get appointment details:**
```javascript
viewAppointmentDetails(appointmentId);
```
