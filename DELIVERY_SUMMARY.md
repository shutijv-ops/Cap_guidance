## 🎯 Quick Start

### For Administrators
1. Click **⚙️ Settings** in the dashboard
2. Adjust the threshold values
3. Click **💾 Save Settings**
4. See success confirmation ✓
5. Settings automatically persist

### For Developers
```javascript
// Get settings anywhere
const settings = loadSettings();
console.log(settings);
// { sessionThreshold: 5, warningThreshold: 10, criticalThreshold: 15 }

// Use in your code
if (studentAppointments > settings.criticalThreshold) {
  showCriticalAlert();
}
```
